# Simple Shop API
A simple e-commerce API where its possible to manage users, addresses, products, carts and orders.

## Tools
This project was created with the following tools:

- Java
- SpringBoot
- Spring Security
- JPA
- MySQL
- JUnit
- Mockito
- H2
- Java Faker
- Swagger
