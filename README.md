## :writing_hand: About

A simple ecommerce API where its possible to manage users and addresses, products, cart and orders.

## :rocket: Tecnologies

This project was created with the following technologies:

- Java
- SpringBoot
- Spring Security
- JPA
- MySQL
- JUnit
- Mockito
- H2
- Java Faker
- Swagger
